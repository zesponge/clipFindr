to run flask backend:
set up python virtual environment using command palette and python version 3.11
cd .venv
Scripts/activate
pip install -r requirements.txt

python -m flask run
or just press the run button after the packages are installed

to run web app front-end:
cd into frontend and summaraizer
npm i
npm start

web app server:
cd into server
npm i
npm start
